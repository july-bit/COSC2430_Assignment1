Marketing and Programming Projects

1. Create a branding search campaign with Google Ads - Samsung

I explored the power of Google Ad in marketing by creating a simulation of a branding campaign for the brand Samsung, focusing on promoting its S7/S7+ Tablet. I created five different ad groups, featuring:

2. Visualizing Citibike Trips with Tableau

I analyzed a data set of train stations in New York City by importing the data set and visualizing data into bar charts, heat maps, pie charts, etc. Through analyzing data visualizations of a csv file, I effectively drew insights on the most popular train stations in New York City, the most popular start and end stations, distribution of genders on the average trip duration, and flow of users at each hour throughout the day.

3. Machine Learning with Python

I utilized a historical dataset from previous loan applications, cleaned the data, and applied different classification algorithms on the data to predict whether a loan case will be paid off or not. I used the following algorithms in a Jupyter Notebook created through an IBM Cloud database:

  • K-Nearest Neighbor
  • Decision Tree
  • Support Vector Machine
  • Logistic Regression

Then I evaluated the accuracy of each algorithm with the following matrics whenever applicable:

  • Jaccard Index
  • F1-Score
  • Log Loss

4. Data Visualization with Python

I examined a customer survey of 2,233 respondents in different data science interests. I used the artist layer of Matplotlib to create a bar chart to visualize the percentage of the respondents' interests. I configured the bar size with, color, and displayed percentages. Next, I created a choropleth map to visualize crime statistics in San Francisco by converting the San Francisco csv file into a pandas dataframe. I used the GeoJSON file to mark the boundaries of the different neighborhoods in the city. The result is a map centered around San Francisco, with a legend title and a correct color scheme.
